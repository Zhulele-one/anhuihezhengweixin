关于和正

安徽和正工程咨询有限公司于2016年12月成立，是一家具有独立法人资格的、新型的现代化企业制公司，企业注册资本500万元人民币。2014年与安徽盛唐工程咨询集团合作成立了安庆分公司。


【人力资源】公司现拥有含一级建造师、中高级专业技术人员8人。
【经营范围】公司主要从事前期项目策划、工程勘察设计、工程造价咨询等业务
【公司信誉】公司严格遵守国家的法律法规,有高的专业水平和客户至上的服务理念
